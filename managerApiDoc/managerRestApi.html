Manager Rest Api 汇总

Manager Rest Api 目录

  • signIn -- 登录接口
  • signUp -- 注册接口
  • userDetail -- 用户信息接口
  • goodsList -- 商品列表接口
  • goodsDetail -- 商品信息接口
  • pay -- 支付接口
  • orderList -- 订单列表接口
  • rushToBuy -- 抢购接口
  • pendingPayment --获取购物车信息
  • payPendingPayment --购物车内商品付款

Api接口返回码说明

  1.     成功：ManagerService.200
  2.     失败：ManagerService.400

Api 调用URL前缀

  • Url:http://49.4.4.124:31935/manager-demo/

Api 接口详细介绍

1）、登录接口

  • rest接口：v1/rest/signIn
  • 请求类别：post
  • 请求参数：
  1. {
  2.     "userName":"jack",
  3.     "userPwd":"123"
  4. }
  • 返回结果：
  1. {
